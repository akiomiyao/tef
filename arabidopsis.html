Transposable Element Finder (TEF) - Detection of active transposable elements from NGS data

[Top Page]
TEF is a high-performance detector of active transposable elements from short-reads of next generation sequencer, based on detection of Target Site Duplication (TSD) at the transposition.
Scripts and Readme: https://github.com/akiomiyao/tef/

Release Notes (TEF v1.6)

Fixed an issue where some indels were incorrectly identified as TE insertion junctions in previous versions.
Added a new filter to exclude these false-positive junctions and reanalyzed the data using TEF v1.6.
Updated outputs to include:
  • reads containing junctions along with their alignments, and
  • files generated by junction_method.verify, which validate junctions based on the alignments.
